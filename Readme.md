## 🌌 About This Repository

This is the source for my **personal developer portfolio** — a black, glassmorphism-styled site with a **cyan-accented UI**, built to showcase my work as a Discord bot developer and the founder of **Sanerex Development**.

The centerpiece of the experience is a custom **loading / intro animation** that plays on first load — setting the tone before the visitor lands on the main portfolio, with smooth transitions into the projects, skills, and contact sections.

## ⚙️ Getting Started

### Prerequisites
Just a modern web browser. No build step required for the base setup.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/bhaws/portfolio.git

# Navigate into the project folder
cd portfolio

# Open index.html in your browser
# or use Live Server in VS Code for hot-reload
```

---

## 🎨 Customization Notes

- Intro animation timing/easing → `assets/css/intro.css`
- Intro animation sequence/logic → `assets/js/introAnimation.js`
- Theme colors (black / glass / cyan) → CSS variables at the top of `style.css`
- Project cards & content → edit directly in `index.html`
